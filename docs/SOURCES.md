# Source notes

Important upstream references:

- Apple Virtualization.framework: VZLinuxBootLoader, VZVirtualMachineConfiguration, VZVirtioFileSystemDeviceConfiguration, VZSingleDirectoryShare, VZVirtioSocketDeviceConfiguration.

- Droidspaces-OSS documentation: Kernel-Configuration.md and Features.md.
- GitHub Actions docs: hosted runners, workflow artifacts, self-hosted runners.

The source tree is intentionally kept out of this repository. CI fetches Linux from kernel.org stable Linux by branch or tag.
